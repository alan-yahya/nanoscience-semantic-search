This web app/API uses custom embeddings from NanoBERT to retrieve relevant Nanoscience papers. It leverages the Hugging Face Inference API and FAISS for a hybrid-vector semantic search.

The application uses embeddings to convert text into vectors, stores them in a FAISS index, and retrieves relevant information based on similarity search.
